# facedemo_qt
这是一个基于linux系统的人脸识别系统演示，通过这个系统展示最近人脸识别系统取得的一些进展。

一、环境部署
-----------------------------
 目前本人测试环境为Ubuntu14.04 其它系统暂未进行测试，若出现任何问题，可与我联系并进行bug修复。
 
 (1)opencv安装： 
~~~cpp
      sudo apt-get install libcv-dev
~~~
 (2)摄像头测试：
 
    由于很多朋友，使用的虚拟机，可能出现读取不到摄像头的情况。为了确保摄像头可用，您可以使用cheese命令测试摄像头。
    
  (3)注意
   
    程序执行一定要在 root 权限下执行，否则 文件创建可能失败，导致系统无法正常运行。
 
二、执行FaceDemo_all程序即可
------------------------------
~~~cpp
  ./FaceDemo_all
~~~

三、测试结果
-------------------------------
 在Ubuntu14.04 虚拟机系统下，测试硬件配置 i7-4790 3.6GHZ
 
   平均测试单张640*480分辨率图像，单次识别共耗时约为30ms，可达到实时标准。
   
 当然，为了各位测试方便，本人也将运行时间以log的形式输出出来，方便各位在自己的机器上进行测试。日志目录为
   result/log_time.txt
